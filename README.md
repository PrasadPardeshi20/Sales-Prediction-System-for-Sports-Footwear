## Overview
- Dataset: ~30,000 records of global footwear sales
- Task: Sales / Revenue Prediction
- Model: Decision Tree Regressor
- Deployment: Streamlit (demo application)

---

## Workflow
- Data cleaning and exploratory data analysis (EDA)
- Feature selection for deployment
- Model training and retraining
- Model evaluation on unseen data
- Interactive prediction using Streamlit

---

## Streamlit App
The Streamlit application allows users to input pricing and customer details
and receive real-time sales/revenue predictions.
